# Obby Revival Project

What is Obby Revival Project (short for ORP)?

Obby Revival Project is a godot game, trying to make obbying outside of roblox, due to recent roblox changes.

Features:
Linking with Roblox Studio
Client where you can play levels
User Generated Levels
Fun physics to mess around with

You can pull request if you want, we appreciate all help :D


Currently, the editor is being reworked, [here's](https://github.com/Dankinations/ORP-RBLXLINK) the link to the roblox studio helper plugin, contribute if u want!
Launcher being worked on [here](https://github.com/Dankinations/ORP-Launcher)
